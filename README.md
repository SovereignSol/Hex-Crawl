# HexCrawl 3D (Props Anywhere + Clean Grid)

## What you asked for
- Place reusable 3D assets anywhere (free placement) or snapped to hex centers
- Perfect-looking hex grid (no triangle gaps between tiles)

## Install & Run
npm install
npm run dev

## Tiles (optional)
Put PNGs in public/tiles/ with exact names:
Forest_1.png, Forest_2.png, Plains_1.png, Plains_2.png, Plains_3.png, Water_1.png, Rocky_1.png

## Models
Put GLB files in public/models/
Then reference as /models/yourModel.glb in the Prop Library.

## Placement
Prop Tool -> CTRL + left click to place.
Toggle snap for hex-center snapping.

Select prop -> arrows move, Q/E rotate, +/- scale, Delete removes.
